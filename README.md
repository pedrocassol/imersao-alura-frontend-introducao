# Projeto Inicial - Imersão Front-End Alura

Este projeto foi desenvolvido como uma introdução à Imersão Front-End da Alura.

## Acesse o projeto

👉 https://pedrocassol.github.io/imersao-alura-frontend-introducao/

## Sobre o projeto

O objetivo desta atividade foi aprender a utilizar ferramentas de Inteligência Artificial no desenvolvimento web.

O site foi gerado com auxílio de IA, utilizando HTML e CSS.

## O que eu aprendi

* Como utilizar IA para gerar código
* Estrutura básica de um site (HTML + CSS)
* Importância de revisar e entender o código gerado

## Observação

Este é um projeto introdutório, com forte uso de IA.
Os próximos projetos da imersão terão maior participação manual no desenvolvimento.

## Tecnologias utilizadas

* HTML
* CSS
* IA (VS Code)
